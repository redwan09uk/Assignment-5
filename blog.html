Welcome to DOM Q&A Session

  1. What is the Document Object Model (DOM)?

    2. Answer:DOM is a Programming interface for HTML and CSS documents.By using DOM we can easily add or change the current HTML or CSS.

    3. Answer:I can select an element from the DOM by using id , class , tagName,classSelector ,attributeSelector , childSelector ,parentSelector etc.
